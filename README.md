# Fake Coin Problem

### Тема:

Задача о фальшивой монете

---

# Описание

Данная работа посвящена классической задаче дискретной математики и теории информации — задаче о фальшивой монете.

Среди набора одинаковых на вид монет одна является фальшивой. Она может быть либо легче, либо тяжелее настоящих монет. Необходимо определить фальшивую монету за минимальное количество взвешиваний на рычажных весах без гирь.

В проекте реализованы:

* моделирование рычажных весов;
* алгоритмы поиска фальшивой монеты;
* дерево решений;
* визуализация стратегий;
* информационный анализ задачи;
* построение решений для нескольких вариантов задачи.

---

# Реализованные задачи

## 1a. Задача (N = 12, n = 3)

* 12 монет;
* одна фальшивая;
* неизвестно:

  * легче она,
  * или тяжелее.

Необходимо за 3 взвешивания:

* найти монету;
* определить ее тип.

---

## 1b. Задача (N = 39, n = 4)

* 39 монет;
* одна фальшивая;
* неизвестно:

  * легче она,
  * или тяжелее.

Необходимо решить задачу за 4 взвешивания.

---

## 2. Задача (N = 27, n = 3)

* 27 монет;
* заранее известно, что фальшивая монета легче.

Необходимо найти фальшивую монету за 3 взвешивания.

---

# Теоретическая основа

Проект основан на:

* теории информации;
* понятии энтропии;
* деревьях решений;
* рекурсивных алгоритмах;
* информационно-оптимальных стратегиях.

Количество информации определяется энтропией:

$$
H = -\sum p_i \log_2 p_i
$$

Для задачи о фальшивой монете каждое взвешивание дает три возможных исхода:

* левая чаша тяжелее;
* правая чаша тяжелее;
* равновесие.

Поэтому за (n) взвешиваний можно различить:

$$
3^n
$$

различных состояний.
